# GlideAI iOS Build

This branch packages GlideAI as a native iOS shell with Capacitor. The app
loads the static `out/` web build inside the iOS project and keeps the
MediaPipe pose model files bundled with the app, so the analysis runs on the
phone without a backend or CDN.

## Build on a Mac

1. Install Xcode and open it once so command line tools are configured.
2. From this repo, install dependencies:

   ```bash
   npm ci
   ```

3. Build and sync the bundled web app into the iOS project:

   ```bash
   npm run ios:prepare
   ```

4. Open the native project:

   ```bash
   npm run ios:open
   ```

5. In Xcode, select the `App` target, choose your signing team, connect an
   iPhone, then run the app.

## Notes

- The iOS bundle id is `com.glideai.mobile` in `capacitor.config.ts`.
- Camera access is declared in `ios/App/App/Info.plist`.
- The copied web bundle under `ios/App/App/public` is generated by
  `npm run ios:prepare` and is intentionally ignored by git.
- Publishing through the App Store requires Apple Developer Program
  membership. Running on your own device for development can use Xcode signing
  with an Apple account.
